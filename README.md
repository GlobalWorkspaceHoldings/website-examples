# Website Styling Examples

A comprehensive visual reference of modern website components and styling patterns. Organizations can copy and paste components directly from the included HTML file to quickly build professional websites.

**See it in action:** [sgf.work](https://sgf.work)

## What's Included

One HTML file containing copy-paste ready examples of:

- **Page layouts**: Homepage, landing pages, about pages, legal pages, error pages
- **Hero sections**: Standard, split-column, full-width variants
- **Components**: Navigation, pricing cards, feature grids, testimonials, FAQs, forms
- **Interactive elements**: Accordions, tabs, modals, link shortener interface
- **Design patterns**: Glassmorphism, responsive layouts, accessibility features

## Usage

Open the HTML file in your browser to see all components rendered. Copy the HTML/CSS for any component you want to use in your own project.

All placeholder text uses "Lorem Ipsum" - replace with your actual content before deploying.
